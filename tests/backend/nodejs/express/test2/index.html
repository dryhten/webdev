Hello, first you need to start the webserver:
node bin/www
nodemon bin/www
npm start

Last one works if you have a correct "start" target in package.json. You can also prefix/set the DEBUG env var to test:*/express:* if you want to enable the debug module, e.g.

DEBUG=test:* nodemon bin/www

The files are in backend/nodejs/express/test2/
Then you need to go to http://127.0.0.1:3000

The project was generated with express generator.
