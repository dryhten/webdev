This page has examples for HTML!



HTML document should be between <html> tags. Optional at the beginning: !doctype html.

HTML docs should have head & body. Example for head in src.
There are tags between < and > and attributes of tags.
To escape special chars & lt ; (less than). Look at src.

This is a paragraph. Basic tags:

br = newline
p = paragraph
div = group smth to apply some changes
span = same as div, but inline
h1 = header size 1
h2 = header size 2 ... etc until h6


Special chars (& + special + ;):
lt = <
gt = >
nbsp = space
